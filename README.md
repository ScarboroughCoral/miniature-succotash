
一个 roadmap 排期工具，找出关键路径，以及最短路径

## 功能点
- 创建任务点，以及任务点之间的连线，UI拖拽交互（类似processon、drawio之类的）
- 根据拓扑排序算法可以看到最终推荐
  - 最短时间是多少
  - 如果考虑人力呢
- 需要保存，再次启动app可以看到
## 技术选型

### 前端
1. 前端框架 react
2. UI拖拽交互 react-diagrams
3. 后端算法执行，暂时考虑3种方案
   1. electron node 环境 napi + ffi
   2. wasm
   3. 直接 web 服务
